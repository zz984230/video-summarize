# B站视频摘要助手

一个浏览器插件，可以自动提取B站视频链接并通过大模型生成视频摘要。

## 功能特点

- 🔍 自动识别B站视频页面并提取视频链接
- 🤖 支持多种大模型API（支持自定义base_url）
- 📝 生成视频整体摘要（200-300字）
- ⏰ 分段内容分析（时间戳+中心思想）
- ⚙️ 可配置的大模型设置
- 💾 本地缓存处理记录
- 🔄 支持中断和重试机制

## 安装使用

1. 克隆项目
2. 安装依赖：`npm install`
3. 构建项目：`npm run build`
4. 在Chrome浏览器中加载扩展程序
5. 打开B站视频页面，点击插件图标即可使用

## 配置说明

首次使用需要在插件设置中配置：
- 大模型API的base_url
- API密钥

## 技术栈

- React + TypeScript
- Chrome Extension API
- Webpack构建
