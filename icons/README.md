# B站视频摘要助手 - 图标文件

由于无法直接生成图标文件，请按以下方式创建图标：

## 图标规格要求

需要创建以下尺寸的PNG图标：
- icon16.png (16x16像素)
- icon32.png (32x32像素) 
- icon48.png (48x48像素)
- icon128.png (128x128像素)

## 图标设计建议

1. **主题色彩**: 使用B站的主题色（粉色 #FB7299）作为主色调
2. **图标元素**: 
   - 可以包含"B"字母标识
   - 添加摘要/文档的图标元素
   - 保持简洁现代的设计风格
3. **背景**: 建议使用透明背景或白色背景

## 创建方法

您可以使用以下任一方式创建图标：

1. **在线图标生成器**: 
   - 访问 https://www.favicon.cc/
   - 或 https://www.logaster.com/

2. **图形设计软件**:
   - 使用Photoshop、GIMP等软件设计
   - 导出为PNG格式，确保背景透明

3. **AI生成图标**:
   - 使用AI图像生成工具创建图标
   - 提示词示例: "Bilibili video summarizer icon, pink and white colors, clean modern design, letter B with document symbol"

创建好图标后，将它们放入项目的 `icons/` 目录中。
